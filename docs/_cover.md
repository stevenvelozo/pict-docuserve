# Pict Docuserve

> A documentation viewer built on Pict

Turn any folder of markdown files into a searchable documentation site. Built on the Pict MVC framework as a native single-page application with hash-based routing, Mermaid diagrams, and LaTeX equations.

- **Serve** -- Preview any markdown folder as a doc site with `npx pict-docuserve serve ./docs`
- **Inject** -- Copy app assets into a folder for GitHub Pages or any static host
- **Search** -- Full-text search powered by Lunr via Indoctrinate keyword indexes
- **Catalog** -- Browse cross-repo module documentation from a single sidebar
- **Diagrams** -- Mermaid flowcharts, sequence diagrams, ER diagrams, and more
- **LaTeX** -- Inline and display math equations via KaTeX
