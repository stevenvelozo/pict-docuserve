# Pict Docuserve

> Documentation viewer powered by Pict

A single-page documentation viewer that renders markdown files as a full documentation site. Built on the Pict MVC framework with zero external runtime dependencies.

- **Serve** -- Preview any markdown folder as a doc site with one command
- **Inject** -- Copy app assets into a folder for static hosting
- **Markdown** -- Supports headings, code blocks, tables, links, and more
- **Catalog** -- Browse cross-repo module documentation via Indoctrinate catalogs
