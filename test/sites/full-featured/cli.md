# CLI Reference

## Commands

### `start`

Start the application.

```bash
full-featured start [options]
```

**Options:**

| Flag | Description |
|------|-------------|
| `-p, --port <port>` | Port to listen on (default: 3000) |
| `-d, --debug` | Enable debug logging |

### `build`

Build the project for production.

```bash
full-featured build
```

### `test`

Run the test suite.

```bash
full-featured test [--grep <pattern>]
```
