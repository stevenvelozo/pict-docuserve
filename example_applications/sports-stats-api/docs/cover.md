# Statcaster

> Real-time sports statistics at your fingertips

A read-oriented API for querying historical and live sports statistics. Statcaster aggregates data from multiple leagues and provides a unified query interface for scores, player stats, standings, and schedules.

- **Multi-League** — NFL, NBA, MLB, NHL, and MLS in one API
- **Historical Data** — Season archives going back 20+ years
- **Live Scores** — Real-time game updates via polling or WebSocket
- **Player Stats** — Career and per-game statistics for every player
