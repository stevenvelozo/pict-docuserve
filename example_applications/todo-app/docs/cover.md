# Taskflow

> Simple task management, done right

A lightweight todo application built with plain JavaScript. Taskflow helps individuals and small teams organize work with lists, priorities, and due dates -- without the bloat.

- **Lists** — Organize tasks into custom lists and categories
- **Priorities** — Flag tasks as low, medium, high, or critical
- **Due Dates** — Set deadlines with automatic overdue detection
- **Tags** — Label and filter tasks with flexible tagging
